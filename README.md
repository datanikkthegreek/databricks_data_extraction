# databricks_data_extraction

Solution Accelerator Data Extraction.

- **App** (FastAPI, apx UI): [databricks_app/](databricks_app/) — use `cd databricks_app` for `uv run apx` and [pyproject.toml](databricks_app/pyproject.toml).
- **ETL** (DLT jobs and pipelines): [databricks_etl/](databricks_etl/) — use `cd databricks_etl` for `uv sync` / `databricks bundle deploy`.

See [README_dev.md](README_dev.md) and [README_deploy.md](README_deploy.md) for setup and deployment.
